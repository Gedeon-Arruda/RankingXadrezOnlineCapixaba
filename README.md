# Ranking de Xadrez Online Capixaba

Projeto isolado para publicar uma nova versão do ranking sem alterar o `RankingXadrezJovem`.

## Fontes

- Lichess: `https://lichess.org/team/ranking-de-xadrez-online-capixaba`
- Chess.com: `https://www.chess.com/club/ranking-de-xadrez-online-capixaba`

## Pastas

- `docs/`: site estático e arquivos JSON gerados
- `scripts/generate_data.py`: geração dos dados
- `.github/workflows/generate.yml`: atualização automática

## Publicação em outro link

1. Crie um novo repositório, por exemplo `RankingXadrezOnlineCapixaba`
2. Copie o conteúdo desta pasta para a raiz do novo repositório
3. Ative o GitHub Pages usando a branch `main` e a pasta `/docs`
4. Se quiser, associe um domínio ou subdomínio da outra empresa

Assim o ranking novo fica independente do projeto do Xadrez Jovem.
