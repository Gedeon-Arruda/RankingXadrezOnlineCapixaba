# Ranking de Xadrez Online Capixaba

Projeto independente do ranking online, publicado separadamente do Ranking Xadrez Jovem.

## Fontes de dados

- Lichess: `https://lichess.org/team/ranking-de-xadrez-online-capixaba`
- Chess.com: `https://www.chess.com/club/ranking-de-xadrez-online-capixaba`

## Scripts

- `npm run generate`: gera os JSONs em `docs/`
- `npm run serve`: sobe um preview local
